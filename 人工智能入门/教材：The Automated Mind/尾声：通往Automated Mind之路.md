# 尾声：通往Automated Mind之路

这本书走完了一条完整的路径：

1. **物理层**（[[教材：The Automated Mind/Phase 1：秩序的底座/Vol 01：C++与算力剥削|Vol 01]] — [[教材：The Automated Mind/Phase 1：秩序的底座/Vol 02：多维数据结构|Vol 02]]）：理解计算的物理代价
2. **模型生态层**（[[教材：The Automated Mind/Phase 3：模型生态与Agent实践/Vol AG-01：AI模型全景与选型|Vol AG-01]] — [[教材：The Automated Mind/Phase 3：模型生态与Agent实践/Vol AG-06：Agent持久记忆与智能编排|Vol AG-06]]）：掌握模型分类、选型决策、Agent实战场景、本地部署、记忆缓存与智能编排
3. **语义层**（[[教材：The Automated Mind/Phase 2：语义空间与认知具象/Vol 03：Embedding映射|Vol 03]] — [[教材：The Automated Mind/Phase 2：语义空间与认知具象/Vol 04：RAG与知识图谱|Vol 04]]）：构建语义的数学基础
4. **认知层**（[[教材：The Automated Mind/Phase 4：深水区与几何/Vol 05：智能体记忆(Alaya)|Vol 05]] — [[教材：The Automated Mind/Phase 4：深水区与几何/Vol 06：流形与几何学|Vol 06]]、[[教材：The Automated Mind/Phase 3：模型生态与Agent实践/Vol 08：Agent部署|Vol 08]] — [[教材：The Automated Mind/Phase 3：模型生态与Agent实践/Vol 08.5：AI工程项目实战|Vol 08.5]]、[[教材：The Automated Mind/Phase 4：深水区与几何/Vol 10：潜意识与拓扑图谱|Vol 10]]）
5. **规则层**（[[教材：The Automated Mind/Phase 5：规则重塑/Vol 07：数字城邦秩序(Neuro-Symbolic)|Vol 07]]、[[教材：The Automated Mind/Phase 5：规则重塑/Vol 11：机械可解释性(SAE)|Vol 11]] — [[教材：The Automated Mind/Phase 5：规则重塑/Vol 12：多智能体与密码学(ZK-ML)|Vol 12]]）

这本书教给你的不是API，而是一组**思考工具**：

- **它把数据放在哪里？物理上怎么访问？**
- **它把关系编码成什么结构？代价是多少？**
- **它的不确定性来自哪里？能用什么方法约束？**
- **它的边界在哪里？超出边界时是优雅降级还是崩溃？**

剩下的路，看你了。
