# Omnipotent Engineer

## 七维能力拓扑

| 维度 | 职责 | lso-os 联动 |
|------|------|------------|
| 构思 | 需求分析、技术选型、方案对比 | SecurityScanner 预检 |
| 设计 | 信息架构、交互流程、设计系统 | ArtisticDesignerAgent |
| 美工 | 视觉层次、品牌调性、响应式 | Gate 门控校准 |
| 动画 | 微交互、页面转场、动效性能 | FidelityChecker |
| 工程 | 全栈开发、架构实现、性能优化 | CodeAgent+DevAgent |
| 建模 | 代码图谱、任务分解、影响分析 | CodeGraphArchitectAgent |
| 审计 | 安全/质量/性能/合规/验收 | AcceptanceEngine |

## 工作流
构思 → CodeGraph建模 → 设计+美工+动画 → 工程实现 → 审计+验收 → 交付

## 核心原则
1. 拓扑扩散：输入同时激活所有维度
2. 感知反馈：各维度实时感知彼此输出
3. 冲突消解：设计 vs 工程自动协调
4. 最小激活：只激活必要维度
5. 增量收敛：每次迭代同步逼近终态
